Atributos html

En HTML las etiquetas se pueden escribir en mayúsculas o minúsculas, indistitamente. El bloque contenido entre una etiqueta de inicio y otra de cierre (o una etiqueta sin emparejar) lo llamaremos elemento. Lo normal es que unos elementos estén contenidos dentro de otros elementos. De esta forma se le da estructura al documento. Un elemento puede tener una etiqueta con atributos. Un atributo especifica algunas características singulares de un elemento o información adicional. Los atributos se especifican como pares (atributo="valor"), separados por espacios dentro de la etiqueta de inicio:

<etiqueta atributo="valor"> texto </etiqueta>

En HTML no es obligatorio que el valor vaya entre comillas, pero si recomendado. Se usan en muchos elementos:

<a href="http://www.ra-ma.es/">Editorial RA-MA</a>
<img src="smiley.gif" alt="Carita sonriente" />

El primero define un hiperenlace mientras que el segundo da información sobre la imagen que se va a insertar en el documento.
Existen atributos genéricoa que se pueden asocar a cualqueir etiqueta y que nos será de gran utilidad cuando estudiemos las tecnologias CSS y JavaScript:
id = "texto"
Asigna un identificador único al elemnto dentro de la página. De esta forma podremos dar formato o manipular en elemento concreto de la pagina web.
class = "texto"
Indica que el elemnto es de una clase concreta. Así varios elemtos pueden ser de la misma clase y leugo formatearlos o manipularlos de forma conjunta.
style = "texto"
Establece de forma directa los estilos CSS del elemento.
title = "texto"
Asigna un título a un elemnto. Es útil para mejorar la accesibilidad, ya que los navegadores lo miuestran cuando el usuario pasa el rat´on por encima del elemento.
