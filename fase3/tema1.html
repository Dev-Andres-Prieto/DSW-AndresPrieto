Etiquetas básicas html

HTML es la tecnología fundamental de la World Wide Web y ha ido evolucionando hasta nuestros días. Para escribir código HTML solo necesitamos un editor de texto sencillo. Sin embargo, resulta muy recomendable utilizar algún editor especifico que nos facilite la escritura con funciones como autocompletar validacones de sintaxis, etc.

Introducción a HTML5:

HTML es un lenguaje de marcas, es decir, consta de texto, que define los contenidos reales de la pagina web, y marcas especiales (también conocidas como etiquetas o tags) que permiten dar "significado" al textoo contenido, así como indicar algún tratamiento especial sobre dicho texto. La idea básica de los lenguajes de marcado es estructurar el contenido mediante dichas marcas o etiquetas (tags)
Lo que hace el estándar HTML5 es especificar que marcas deben utilizarse para escribir un documento web y que significa cada una de ellas. De esta forma cualquier navegador sabrá cómo interpretarlas.

Por ejemplo:

Todo documento HTML ha de ir delimiado por la etiqueta de inicio y cierre <html> </html>. dentro de estas dos etiquetas tendremos dos partes bien diferenciadas: Cabecera (elemento <head> </head>) y cuerpo (elemento <body> </body>). Estas son las etiquetas HTML que encontraremos en cualquier pagina web y que definen su estructura.
